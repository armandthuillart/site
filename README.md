This is my personal website.

I use it to write, publish, and share things.

## Stack

- **Astro**
- **MDX**
- **Tailwind CSS**
- **Cloudflare Workers**

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview
```

Feel free to open an issue if something looks off or broken. Pull requests are disabled.
