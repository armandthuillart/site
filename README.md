This is a [Astro](https://astro.build/start) project bootstrapped with [`bunx @astro/latest`](https://docs.astro.build/en/install-and-setup/#install-from-the-cli-wizard).

## Getting Started

First, run the development server:

```bash
bun dev
```

Open http://localhost:3000 with your browser to see the result.

You can start editing the page by modifying `content/blog/*.mdx`. The page auto-updates as you edit the file.

## Open Source

This repository is public but read-only. Pull requests are therefore not accepted. If you have suggestions or feedback, please feel free to open an issue instead.
